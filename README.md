# IT1214-Student-Grade-System

This is a simple Java program to manage student marks. It helps teachers to save student details digitally instead of using paper sheets.

## Project Purpose
The purpose of this project is to manage student records easily. It uses basic Java and OOP concepts like encapsulation.

## Features
- Add a student with ID, Name, and Marks
- Display all student details
- Search for a student by ID
- Calculate the average mark of all students

## How to Run the Program

1. Open Command Prompt in the project folder.
2. Compile the Java files:

```bash
javac Main.java Student.java
```

3. Run the program:

```bash
java Main
```
